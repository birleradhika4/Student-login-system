Student Login System (PHP + MySQL)

A simple and attractive Student Login System project developed using HTML, CSS, PHP, and MySQL. This project is suitable for college practical exams and demonstrates basic web development concepts with database connectivity.

#This system allows students to:

->Register themselves
->Login using username and password
->View their profile
->Edit their details
->Change password
->Logout securely
It also includes a modern UI with hover effects, icons, and success messages.

#Technologies Used:
HTML5
CSS3
PHP (Core PHP)
MySQL Database
Font Awesome (for icons)
